## 🔨 Development

### 👀 Prerequisites

- [ollama](https://ollama.dev/download)


### 🚀 Setting up

```sh
make setup
```

## ⚡️ Generate Commit messages

In any git directory with staged changes, run:

```sh
ai-commit
```


## ✨ Linters and Formatters

Check for linting rule violations:

```sh
make check
```

Auto-fix linting violations:

```sh
make fix
```

## 🤸‍♀️ Getting Help

```sh
make

# OR

make help
```
